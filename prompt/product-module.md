# Product

- Resource Name: Product
- Parent Menu: None

## List Page

- Table Properties: Product Name, Quantity (Qty), Status (Available, Not Available)
- Filter: Status

## Detail Page
- Section 1: General Information
  - Fields: Product Name, Quantity (Qty), Status, Created At, Updated At
- Section 2: Product Completeness
  Display as a table with the following properties:
  - Name (e.g., Charger, Keyboard, etc.)
  - Note (additional information about the item)

## Form

- Section 1: General Information
  - Fields: Name, Status, Status

- Section 2: Product Completeness
  - Display as a table with the following properties:
    - Name (e.g., Charger, Keyboard, etc.)
    - Note (additional information about the item)
  - User can add, edit, or delete existing product Completeness
